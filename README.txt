----------

This Language Pack contains the German language files for Elgg 1.7.14/1.7.15 including the core plugins.

To install / update the language pack simply copy the files into the corresponding folders: the main language file into the "language" folder in your Elgg root folder and the plugins' language files into the corresponding subfolders in the "mod" folder. This German language pack retains this folder structure. Therefore, you can just copy the files into your Elgg root folder.

The language used on your Elgg installation can either be set for the whole site via Site Administration ("Administration Deiner Community-Seite") or each user can set the language individually via "Change your settings" ("Account konfigurieren") available in the Settings menu ("Einstellungen").


----------

Dieses Sprachpaket enthält die deutschen Sprachdateien für Elgg 1.7.14/1.7.15 inklusive der mitgelieferten Plugins.

Für die Installation / zum Updaten müssen die Dateien einfach in die entsprechenden Verzeichnisse kopiert werden: die Hauptsprachdatei in das "language"-Verzeichnis im Elgg-Rootverzeichnis und die Sprachdateien der Plugins in die entsprechenden "language"-Unterverzeichnisse der jeweiligen Plugins im "mod"-Verzeichnis. Diese Verzeichnisstruktur wurde in diesem Sprachpaket beibehalten. Es ist also möglich, die mitgelieferten Dateien einfach in das Elgg-Rootverzeichnis zu kopieren.

Die Festlegung der verwendeten Sprache kann entweder global für die gesamte Elgg-Installation über Site Administration ("Administration Deiner Community-Seite") vorgenommen werden oder jeder Benutzer kann individuell die zu verwendende Sprache über das Settings-Menu ("Einstellungen") unter dem Punkt "Change your settings" ("Account konfigurieren") festlegen.
